# Windows Golden Image Build

## Features
- Unattended Windows install
- Chrome + Steam install
- GPU auto-detection (NVIDIA/AMD/Intel)
- Optimised for lab / home use

## Usage
1. Copy autounattend.xml to USB/ISO
2. Boot machine
3. Install runs automatically

## Build Process
- VM build
- Sysprep
- Capture WIM
- Replace ISO install.wim
